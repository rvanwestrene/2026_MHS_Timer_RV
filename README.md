# 2026_MHS_Timer
A javascript timer for school.

This was originally vibe coded using Gemini.

The goal is to have a timer which counts down to the end of a period that aligns with our rather funky bell times.  The timer changes colour towards the end of each period so that students (and teachers) know when it is time to wind down / do an exit task.

It only works on Chrome / Edge and not Firefox as I wanted it to stay on the top of other windows so that it is always visible.
